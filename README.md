# HR_Analytics_Dashboard

## 🚀 Overview
This project focuses on analyzing employee attrition and performance using Power PI.  
The goal was to identify key factors affecting employee turnover and present insights through an interactive and easy-to-understand dashboard.

## 📌 Key Features
- Attrition analysis by department, job role, and age group  
- Performance and job satisfaction insights  
- Analysis of salary and work-life balance impact  
- Summary KPIs (Total Employees, Attrition Count, Active Employees, Avg Salary)

## 📈 Insights
- Attrition is mainly influenced by low salary and low job satisfaction  
- Poor work-life balance increases the likelihood of employee turnover  
- Younger employees and analysts show higher attrition rates  
- Departments like Finance and IT require more attention  

## 🎯 What I Learned
- How to analyze HR data effectively  
- Turning raw data into meaningful insights  
- Designing clear and interactive dashboards  
- Improving problem-solving and analytical thinking
